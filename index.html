    How To

    Use PowerShell (Windows) to retrieve:

    • OS name, version, 32/64-bit
    • CPU model + cores/threads
    • RAM modules: size + speed
    • Storage: model + type (NVMe/SATA) + media type + size
    • GPU

    Windows — PowerShell commands

    OS (name/version/architecture) 
    Get-ComputerInfo | Select-Object WindowsProductName, WindowsVersion, OsArchitecture
    CPU (model + cores/threads) 
    Get-CimInstance Win32_Processor | Select-Object Name, NumberOfCores, NumberOfLogicalProcessors
    RAM (modules size + speed) 
    Get-CimInstance Win32_PhysicalMemory | Select-Object Manufacturer, Speed, @{n="CapacityGB";e={[math]::Round($_.Capacity/1GB,2)}}
    Storage (model + size) 
    Get-PhysicalDisk | Select-Object FriendlyName, MediaType, BusType, @{n="SizeGB";e={[math]::Round($_.Size/1GB,2)}}
    GPU 
    Get-CimInstance Win32_VideoController | Select-Object Name
